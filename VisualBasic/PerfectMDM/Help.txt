Термины и определения
Контрагент - Каждый из партнёров, заключающих контракт с вашей компанией. 
Персона - физическое лицо - являющееся представителем контрагента
Юридическое лицо аффилированое с контрагентом их может не быть (в случае наличных взаиморасчетов, а может быть несколько)
Заказ - перечень продукции, которую заказал клиент. Карточка заказа - визуальное отображение содержимого заказа
Жизненный цикл заказа - последовательность сменяемых состояний заказа от Расчета до выдан
Признаки заказа - Заказ закрыт, проставляется сотрудником, ответственным за финансовые показатели компании

Кальуляция
Профилирование настройка калькулятора в части выполнения тех или иных технологических операция необходимых для производства выбранной продукции
Конфигурирование настройка калькулятора для определения типов материалов, доступных для выполнения той или иной продукции, например, чтобы при выборе печати на конвертах были доступны только конверты


Производство 
Технологическая операция совокупность действий совершенных сотрудником компании в целях производства продукции на его рабочем участке
Технологическая карта заказа - перечень операций, необходимых для выполнения заказа


Описание работы универсального прайса.
Универсальный прайс состоит и формы калькулятора, формы установки цен и таблиц Базы данных.
Таким образом вс содержимое универсального прайc-листа хранится в базе данных.
При открытии окна универсального прайса в заголовке окна указывается какому направлению он соответствует.
Структура его следующая:

               Направление
			  /           \
             /             \
  Группы прайсов           Текстовый параметр (кол-во с листа), param2
  /         \    \                             \    
Наценки      \    Наименования прайсов---------- Цена
              \                               /   
          Цифровой параметр(количество)______/

Название текстового параметра хранится в поле description таблицы Services (Направления)
если он там указан то его видно в форме прайс листа и калькуляторе

Нумерация
поле numbering в параметрах
1 - нумерация каждый день с 1
2 - сквозная каждый год с 1
3 - нумерация каждый месяц с 1

При формировании документов тип продукции 
1- товары
2- услуги

Для отображения в суффиксе номера заказа буквы или цифры отличающего данный заказ от заказов других офисов
в таблице Offices в поле description должно быть введен данный набор символов

Наценка к стоимости при использовании не базоваого материала
SELECT ParamValue FROM PriceServices WHERE Section = 'VisitCard' AND ParamName = 'VDneBazMat'

Настройка калькуляторов
* В этом выпадающем списке будут находится материалы
** В этом выпадающем списке будут типы материалов 

Для универсальных калькуляторов в названии услуг которых есть У-1 в поле Описание/параметр для универсального калькулятора
содержится число полезных прогонов, включенных в базовую стоимость приладки.
